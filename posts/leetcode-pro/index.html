Prefix-Suffix pattern#

In many problems, when we have to find something like number of triplets (x, y, z) on some conditions, we can use prefix-suffix pattern.

general concept#

n = len(arr)
prefix = [0] * (n+1)
suffix = [0] * (n+1)
ans = [0]*n

# calculating prefix
for i in range(1, n+1):
    prefix[i] = # calculation

# calculating suffix
for i in range(n, -1, -1):
    suffix[i] = # calculation

# in many cases, we can do calculation in single loop
# but, it always easy to calculate in two loops

for i in range(n):
    ans[i] = combine(prefix[i], suffix[i+1])

problems#

2179. Count Good Triplets in an Array

Hint in triplet (x, y, z), focus on y and find #x and #y
Code 
from sortedcontainers import SortedList
class Solution:
    def goodTriplets(self, nums1: List[int], nums2: List[int]) -> int:
        ridx = {x:idx for idx, x in enumerate(nums2)}
        n = len(nums1)
        
        ans = 0
        prefix = SortedList()
        suffix = SortedList(range(n))
        
        for _, x in enumerate(nums1):
            idx = ridx[x]
            pi = prefix.bisect_left(idx) # calculating prefix[i]
            si = len(suffix) - suffix.bisect_right(idx) # calculating suffix[i]
            prefix.add(idx)
            suffix.remove(idx)
            ans += pi * si # combine(pi, si)
            
        
        return ans

Interval#

general concept#

Many problem contains time intervals. Generally, many interval problem can be solved greedily. by sorting intervals and then calculating something.

intervals.sort(customfunction)

stk = []
for (s, e) in intervals:
    if stk and overlap(stk[-1], (s, e)):
        # do something
    else:
        stk.append((s, e))

problems#

1288. Remove Covered Intervals

Hint sort by (start, -end)
Code 
class Solution:
    def removeCoveredIntervals(self, intervals: List[List[int]]) -> int:
        intervals.sort(key=lambda x: (x[0], -x[1]))
        
        stk = []
        for x, y in intervals:
            if stk and stk[-1][1] >= y and stk[-1][0] <= x:
                continue
            else:
                stk.append([x, y])
        return len(stk)
